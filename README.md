# Personal Finance Tracker Dashboard

A simple finance dashboard built using Python and Streamlit to track income, expenses, savings, and monthly financial insights.

## Live Demo

👉 [Open Personal Finance Tracker](https://mental-health-tracker-7vfw9vpwzxqyyphbjqe5bm.streamlit.app/)

## Features

- Add income and expense transactions
- Track remaining balance
- Prevent overspending
- Detect mismatched transaction descriptions
- View transaction history
- Delete individual transactions
- Clear all transactions
- Download financial report as CSV
- Monthly savings and expense analysis
- Interactive charts and visualizations

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run

```bash
git clone https://github.com/NehaJOShi259/Personal-Finance-Tracker.git
cd Personal-Finance-Tracker
pip install -r requirements.txt
streamlit run finance_dashboard.py
```
